# Austo Automobiles Data Analysis

## Project Overview
Austo Motor Company is a leading automobile manufacturer specializing in SUV, Sedan, and Hatchback models. In a recent board meeting, concerns were raised regarding the effectiveness of the current marketing campaign. The company decided to bring in an analytics professional to analyze customer data and improve the marketing strategy.

## Business Problem
The objective of this project is to analyze customer and automobile data to understand customer demand patterns. These insights can help the company enhance customer experience and optimize its marketing strategies.

## Tools & Technologies
Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Jupyter Notebook

## Key Steps Performed
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Visualization of relationships between variables  
- Identification of key demand patterns and insights  

## Dataset
The Austo Automobiles dataset contains information about automobile features, customer preferences, and purchasing patterns.

## Outcome
The analysis provides valuable insights into customer demand and behavior, helping the company make data-driven decisions to improve marketing campaigns and customer experience.
